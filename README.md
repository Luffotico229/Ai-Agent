# AI Agent 🤖

An AI agent built in Python that connects to Google's Gemini API
to process and respond to queries autonomously.

## 🛠️ Technologies
- Python
- Google Gemini API

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Gemini API key to a `.env` file
4. Run: `python main.py`
